# Welcome to Wzrd Foundation

Welcome to the official GitHub organization for the WZRD Foundation.

Wzrd Foundation is the open infrastructure layer behind Trade Wzrd - focused on building execution tooling, automation frameworks, and unified trading infrastructure across platforms.

Our goal is simple:

Build open, extensible infrastructure for automated trading.

* Website: [https://tradewzrd.com](https://tradewzrd.com)
* Docs: (Coming Soon)
* [Discord](https://discord.gg/efqSWQTCu2)

## What We’re Building

Wzrd Foundation will host the core open-source components powering the broader TradeWzrd ecosystem, including:

* Wzrd API (Unified execution API layer)
* Automation SDKs
* Webhook frameworks
* Risk & execution engines
* Cross-platform trade routing tools
* Example integrations and starter kits

This organization is where infrastructure lives.

---

## Key Repositories (Upcoming)

* **wzrdapi**
  Unified API layer for routing, execution, and automation across supported trading platforms.

* **automation-examples**
  Example projects showing how to connect TradingView, custom bots, and third-party systems to Wzrd infrastructure.

* **wzrd-sdk**
  Developer SDK for interacting with WZRD execution and automation services.

* **wzrd-docs**
  Developer documentation, integration guides, and API references.

* **awesome-wzrd**
  Curated tools, community builds, templates, and extensions built on top of WZRD.

---

## Vision

Most trading platforms are fragmented.

Different brokers.
Different APIs.
Different execution rules.

WZWzrdRD Foundation aims to standardize the execution layer - making it easier for developers and traders to build automation without rewriting infrastructure for every platform.

One execution layer.
Multiple platforms.
Unified control.
